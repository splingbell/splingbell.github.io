# TigerQueen 2D Character Package

SpriteSkin対応Prefabと4方向Walkアニメーションがセットになった2Dキャラクターパッケージです。  
Prefabをシーンに配置するだけで、すぐにアニメーション付きキャラクターを使用できます。

---

## 概要

本パッケージには、SpriteSkin対応のPrefabと4方向のWalkアニメーションがセットになった2Dキャラクターが含まれています。  
リグ設定済みのため、自由にアニメーション追加が可能です。

2Dゲームやアニメーション表現を必要とするプロジェクトで、手軽に導入できます。

---

## パッケージ内容

- PSDスプライトデータ  
- SpriteSkin対応Prefab  
- Walkアニメーション（4方向）
  - Forward
  - Back
  - Left
  - Right  

---

## 対応環境

- Unity 2019以降  
- Universal Render Pipeline (URP)
- 2D Animation Package
- 2D PSD Importer

---

## 使用方法

1. `TigerQueen.prefab` をシーンにドラッグ＆ドロップ  
2. Animationコンポーネントで再生したいWalkアニメーションを選択  
3. そのままゲーム内キャラクターとして使用可能
